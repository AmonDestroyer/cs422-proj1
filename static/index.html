Introduction

As the impact of climate change becomes increasingly apparent, understanding and predicting future weather patterns has become crucial. A web app focused on climate cast can provide users with valuable insights into climate trends and predictions for their location. To achieve this, the app will use tools from Data Science and Machine Learning, particularly time series forecasting, to derive models that can forecast future weather patterns. By leveraging the power of AI and ML, this web app can provide an accessible and user-friendly way for people to gain insights into the future of their local climate.

Robust Database

The web app will have a database with a strong backend to handle large amounts of data, and will be designed to support efficient queries and data management.

Multiple User Roles

The web app will support three distinct user roles: Contributor, MLE/DS, and DB Admin. Each role will have its own set of permissions and capabilities.

Data Upload

The Contributor user will be able to upload training and test data in CSV format, with options for specifying the date range and time interval of the data.

Upload Training Data

Data Download

A DS/MLE downloads data from the repository, formulates and trains a forecasting model, predicts the results according to the forecasting tasks, and uploads the results and related data.

Download Data

Solution Upload

The MLE/DS user will be able to upload their solution in CSV format, which will be compared to the test (forecast) data to generate an error response.

Propose Solution

Data Management

The DB Admin user will have access to a backend management tool that will allow them to view, add, edit, or delete data in the database using SQL queries.

Manage Data

Error Logging and Reporting

The web app will have a system for logging errors and generating reports to help diagnose and fix issues with the app or data.
